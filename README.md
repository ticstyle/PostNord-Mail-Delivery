# PostNord Mail Delivery
Swedish PostNord Mail Delivery integration for Home Assistant 

To add this integration please add custom repository https://github.com/ticstyle/PostNord-Mail-Delivery/ to HACS in your Home Assistant.

## Installation

[![](https://my.home-assistant.io/badges/hacs_repository.svg)](https://my.home-assistant.io/redirect/hacs_repository/?owner=ticstyle&repository=PostNord-Mail-Delivery&category=Integration)

Via [HACS](https://hacs.xyz/) or manually copy `postnord_mail_delivery` folder from [latest release](https://github.com/ticstyle/PostNord-Mail-Delivery/releases/latest) to `custom_components` folder in your config folder.
